Kernel axioms
∧
Conjunction, not addition. A∧B produces emergence C. C is not the sum of A and B — it is the state that arises when both are simultaneously present. This is not ring arithmetic.
∇
Remainder is signal, never waste. 0 ≠ 1 is enforced in the remainder register. What the Bull discards is what the spine accumulates. Landauer's principle applies: every erasure costs.
⊘
Shadow is memory. The Shadow is the cross-epoch credit assignment register — what this conjunction promises the next layer it cannot deliver through C alone.
↻
Operations are cyclic. Phase lock (∇ = 0) marks clean succession. Open remainder means the wave must complete before the next charge begins. The trough is not optional.

Why should an LLM trainer care?
Analogies to neural network phenomena
Grokking / modular arithmetic
Remainder as failure to generalize
When transformers grok modular addition, they learn to suppress ∇ — to find a representation where the next multiple is phase-locked. Before grokking, the remainder register is open. After: locked. The phase transition IS the grokking event.
SAE / Dictionary learning
Shadow as orthogonalized features
Sparse autoencoders recover features that superposition erased. The shadow ⊘ is the structure that lived in the activation before the attention head's winner-takes-most or collapsed it. Dictionary atoms are reconstructed shadows.
Token / embedding addition
Naive and vs. real interference
Residual stream addition assumes clean superposition (remainder = 0). Real feature composition has ∇ ≠ 0: two features whose AND isn't phase-locked create interference that later layers must absorb. That interference is the remainder accumulating as noise.
Induction heads / successor circuits
Phase lock as clean succession
An induction head succeeds when A ∧ context → next token with ∇ = 0. When the circuit is incomplete, the remainder is nonzero: the head is guessing. Remainder ∇ measures the head's uncertainty as a first-class signal, not just a loss number.
Cross-layer credit assignment
Shadow as inter-layer promise
The shadow ⊘ of a conjunction is what this layer promises the next one without being able to deliver it through C alone. Backprop is one way to honor that promise. The spine accumulates unpaid shadows as a temporal credit register — a NAND-native alternative to gradients.
Attention / softmax
Or-composition as attention collapse
Softmax is or applied to all query-key scores: max wins, everything else becomes the suppressed remainder. Uniform attention (diffuse) is the only state where no remainder is generated. Sharp attention is high-∇ composition — information is concentrated and some is lost.

Training exercises
Six problems — from observation to invention
  • 01 Phase lock hunt. Find two values A and B where A and B gives remainder = 0. What is special about these pairs? Try 0, 1, primes.
  • 02 Remainder accumulation. Find a chain A and B and C where the total remainder is greater than any single-step remainder. What does this tell you about depth in neural networks?
  • 03 Same emergence, different shadow. Find two different pairs that produce the same emergence C but different shadows ⊘. What does this mean for reconstruction / SAE?
  • 04 Shadow stabilization. Run a long chain of conjunctions. Does the shadow ⊘ converge to a stable value? Under what conditions? This mirrors induction head convergence.
  • 05 Identity elements. What happens when you and with 0? With 1? With itself (A and A)? Which of these is a true identity? Which generates a remainder?
  • 06 ★ Invent an operator. Design a custom conjunction rule that keeps remainder = 0 more often than and, but is not trivially phase-locked. What must it sacrifice? Try to express it as: C = f(A,B), ∇ = g(A,B), ⊘ = h(A,B).
